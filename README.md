# Gamified Environmental Education Platform

## Overview

The Gamified Environmental Education Platform is an interactive web application designed to educate children about environmental sustainability through engaging lessons and educational games. The platform combines gamification with learning to make environmental education enjoyable, interactive, and effective.

Users can complete lessons, play educational games, answer quizzes, earn points, unlock achievements, and improve their leaderboard rankings while gaining knowledge about environmental conservation.

---

## Objectives

- Promote environmental awareness among children.
- Encourage learning through interactive gameplay.
- Make environmental education fun and engaging.
- Improve knowledge retention using gamification techniques.
- Motivate users with rewards, achievements, and leaderboards.

## Installation

Clone the repository

```bash
git clone https://github.com/Tamil-arasan-M-git/Gamified-Environmental-Education-Platform.git
```

Navigate to the project directory

```bash
cd Gamified-Environmental-Education-Platform
```

Install frontend dependencies

```bash
cd frontend
npm install
```

Install backend dependencies

```bash
cd ../backend
npm install
```

Run the frontend

```bash
npm start
```

Run the backend

```bash
npm run dev
```
